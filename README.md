# ERC20 Token in pure Yul

This is an implementation of the [eip-20](https://eips.ethereum.org/EIPS/eip-20) specification in pure Yul.

## Running tests

Make sure you have foundry and the solidity compiler installed

Run tests:

```bash
forge test -vv
```
